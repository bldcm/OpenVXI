VXIscResult GetDTMF

(struct VXIscInterface* pThis,
  VXIbyte* dtmfDigit)

Read a single DTMF digit from the DTMF queue
Warnings: There is only one DTMF queue and it is only accessible via this routine and FlushDTMFQueue( ). In systems with multiple clients, they must negotiate among themselves for access to the queue.

Parameters:
dtmfDigit - [OUT] Returns the DTMF digit, returned as an ASCII character (0 - 9, *, #, A - D)
Returns:
VXIsc_RESULT_SUCCESS on success
VXIsc_RESULT_NON_FATAL_ERROR no DTMF digit available
