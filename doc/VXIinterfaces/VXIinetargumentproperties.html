VXIinet argument properties

Keys identifying properties in VXIMap for Prefetch( ) and Open( )
INET_CACHE_CONTROL_MAX_AGE
Value for the HTTP 11 Cache-Control max-age directive for requests.
INET_CACHE_CONTROL_MAX_STALE
Value for the HTTP 11 Cache-Control max-stale directive for requests.
INET_CACHING
Type of caching to apply: safe or fast.
INET_OPEN_IF_MODIFIED
Conditional open, used for cases where a cache outside of VXIinet is being maintained and the desired object is already present in that cache, but the user needs to verify whether that cached object is valid for reuse or not.
INET_OPEN_LOCAL_FILE
Whether to open local files normally or to return a VXIinet_RESULT_LOCAL_FILE error while still returning the stream information if requested.
INET_PREFETCH_PRIORITY
Prefetch priority.
INET_SUBMIT_METHOD
Submit method: GET or POST.
INET_SUBMIT_MIME_TYPE
Submit MIME type.
INET_TIMEOUT_OPEN
Open timeout.
INET_TIMEOUT_IO
Read timeout.
INET_TIMEOUT_DOWNLOAD
Total download timeout.
INET_URL_BASE
URL base for resolving relative URLs.
INET_URL_QUERY_ARGS
URL Query Arguments.
INET_CLOSE_CONNECTION
This property specifies whether the connection should be closed after the fetch is completed.
INET_NEW_CONNECTION
This property specifies whether a new connection should be established regardless of the state of a previously established connection.
VXIinet functions take a VXIMap argument which contains a set of key/value pairs. The listed arguments must be supported by an implementation. Additional arguments can be added to this argument in other implementations. Time durations are specified in milliseconds, see below for valid values for the enumerated type properties.
