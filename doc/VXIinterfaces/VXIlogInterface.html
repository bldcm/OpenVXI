ContentClose
Close a stream for logging (potentially large or binary) content NOTE: This is only available as of version 1.1 of the VXIlogInterface, use LOG_CONTENT_METHODS_SUPPORTED( ) to determine availability.
ContentWrite
Write (potentially large or binary) content to a logging stream NOTE: This is only available as of version 1.1 of the VXIlogInterface, use LOG_CONTENT_METHODS_SUPPORTED( ) to determine availability.
The VXIlogInterface provides a set of functions which are used for logging by all the OpenVXI browser components.
