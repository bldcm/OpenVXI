VXIasResult TriggerRecord

(struct VXIasInterface* pThis,
  const VXIchar* mimeType)

Triggers delivery of audio input data to all listeners, non-blocking
This only triggers audio input data delivery. This routine is expected to return after data delivery has been enabled (as opposed to blocking until data delivery is complete).

Parameters:
mimeType - [IN] MIME content type for the audio input data, may be NULL or an empty string to indicate that any MIME content type supported by the recognizer is acceptable.
Returns:
VXIas_RESULT_SUCCESS on success
VXIas_RESULT_BAD_MIME_TYPE, MIME type is not supported
