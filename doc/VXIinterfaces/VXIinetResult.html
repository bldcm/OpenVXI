typedef enum VXIinetResult

Result codes for interface methods
VXIinet_RESULT_FATAL_ERROR
Fatal error, terminate call
VXIinet_RESULT_IO_ERROR
I/O error
VXIinet_RESULT_OUT_OF_MEMORY
Out of memory
VXIinet_RESULT_SYSTEM_ERROR
System error, out of service
VXIinet_RESULT_PLATFORM_ERROR
Errors from platform services
VXIinet_RESULT_BUFFER_TOO_SMALL
Return buffer too small
VXIinet_RESULT_INVALID_PROP_NAME
Property name is not valid
VXIinet_RESULT_INVALID_PROP_VALUE
Property value is not valid
VXIinet_RESULT_INVALID_ARGUMENT
Invalid function argument
VXIinet_RESULT_SUCCESS
Success
VXIinet_RESULT_FAILURE
Normal failure, nothing logged
VXIinet_RESULT_NON_FATAL_ERROR
Non-fatal non-specific error
VXIinet_RESULT_NOT_FOUND
Named data not found
VXIinet_RESULT_FETCH_TIMEOUT
URL fetch timeout
VXIinet_RESULT_FETCH_ERROR
Other URL fetch error
VXIinet_RESULT_WOULD_BLOCK
I/O operation would block
VXIinet_RESULT_END_OF_STREAM
End of stream
VXIinet_RESULT_LOCAL_FILE
Local file, told not to open it
VXIinet_RESULT_NOT_MODIFIED
Conditional open attempted, cached data may be used
VXIinet_RESULT_UNSUPPORTED
Operation is not supported
Result codes less then zero are severe errors (likely to be platform faults), those greater then zero are warnings (likely to be application issues)
