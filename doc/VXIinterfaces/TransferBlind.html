VXItelResult TransferBlind

(struct VXItelInterface* pThis,
  const VXIMap* properties,
  const VXIchar* transferDestination,
  VXIMap** resp)

Performs a blind transfer.
Perform a blind transfer into the network. The implementation is generally going to be platform dependant and will require a lot of configuration per installation location.

Parameters:
pThis - the line on which the transfer is to be performed.
properties - [in] termination character, length, timeouts...
transferDestination - [in] identifier of transfer location (e.g. a phone number to dial or a SIP URL).
- data [in] the data to be sent to the transfer target.
resp - [out] Key/value containing the result. Basic keys: TEL_TRANSFER_DURATION - length of the call in ms TEL_TRANSFER_STATUS - see VXItelTransferStatus
Returns:
VXItel_SUCCESS if operation succeeds, VXItel_RESULT_CONNECTION_NO_AUTHORIZATION, VXItel_RESULT_CONNECTION_BAD_DESTINATION, VXItel_RESULT_CONNECTION_NO_ROUTE, VXItel_RESULT_CONNECTION_NO_RESOURCE, VXItel_RESULT_UNSUPPORTED, VXItel_RESULT_OUT_OF_MEMORY, VXItel_RESULT_FAILURE
