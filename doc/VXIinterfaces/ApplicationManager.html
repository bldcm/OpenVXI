Application Manager

VXIplatform interface and testClient
VXIplatform
VoiceXML platform interface
The application manager consists of a system operator user interface to start, stop, and configure the platform as well as a VXIplatform interface implementation that interacts with the OpenSpeech Browser components for initialization and servicing calls.

The OpenSpeech Browser provides a basic application manager, called testClient, built on top of a reference VXIplatform interface implementation called SBclient. These components are provided in source form for integrators as a getting started toolkit. Integrators can use all or none of this code in integrating the OpenSpeech Browser PIK into their platforms. See the src/client directory in the distribution.
