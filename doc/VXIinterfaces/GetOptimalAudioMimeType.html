VXIapResult GetOptimalAudioMimeType

(struct VXIapInterface* pThis,
  VXIchar* contentType,
  const VXIint contentTypeLen)

Retrieve the preferred MIME content type for play operations
This retrieves the preferred MIME content type for play operations, allowing clients to obtain that type whenever possible from audio data sources such as text-to-speech engines. Note that this value is only a hint, the client will play audio with other MIME content types in many other such as when the application developer specifies an audio file.

Parameters:
- contentType [OUT] Preferred MIME content type for audio
contentTypeLen - [IN] Size, in characters, of contentType
Returns:
VXIap_RESULT_SUCCESS on success
