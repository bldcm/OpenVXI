VXIasResult TriggerStop

(struct VXIasInterface* pThis)

Stops the record operation
This routine is synchronous and returns when the record operation is either stopped or there is an error. If success is returned, an AS_EVENT_STOPPED event must be delivered before this returns.

Returns:
VXIas_RESULT_SUCCESS on success
VXIas_RESULT_NON_FATAL_ERROR if no record is in progress
Another VXIas_RESULT_[...] code for severe errors
