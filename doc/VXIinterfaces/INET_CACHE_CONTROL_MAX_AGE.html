#define INET_CACHE_CONTROL_MAX_AGE

Value for the HTTP 11 Cache-Control max-age directive for requests.

Documentation

Value for the HTTP 11 Cache-Control max-age directive for requests. This specifies the client is willing to accept a cached object no older than this value (given in seconds). A value of 0 may be used to force re-validating the cached copy with the origin server for every request. In most cases, this property should not be present, thus allowing the origin server to control expiration. Value is a VXIInteger
