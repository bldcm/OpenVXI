VXIPLATFORM_API VXIplatformResult VXIplatformCreateResources

(VXIunsigned channelNum,
  VXIMap* args,
  VXIplatform** platform)

Creates a VXIplatform object for the specified channel number.

Documentation

Creates a VXIplatform object for the specified channel number. This creates the VXItel, VXIprompt and VXIrec objects associated with this channel.

Parameters:
channelNum - [in] The channel number of the platform to be created.
args - [in] Configuration arguments. Implementation dependant.
platform - [out] A pointer to the platform to be allocated.
