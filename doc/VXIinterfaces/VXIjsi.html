VXIjsiInterface
VXIjsi interface for ECMAScript evaluation
Abstract interface for interacting with a ECMAScript (JavaScript) engine. This provides functionality for creating ECMAScript execution contexts, manipulating ECMAScript scopes, manipulating variables within those scopes, and evaluating ECMAScript expressions/scripts.

There is one ECMAScript interface per thread/line.
