Object interface
OBJECT_CLASS_ID
Keys identifying properties in VXIMap for Execute( ) and Validate( )
OBJECT_VALUE
Keys identifying properties in the VXIMap for an individual parameter ( element) in cases where "valuetype" is not "data"
VXIobjectInterface
VXIobject interface for executing VoiceXML objects
Abstract interface for VoiceXML object functionality that allows integrators to define VoiceXML language extensions that can be executed by applications through the VoiceXML object element. These objects can provide almost any extended functionality that is desired.

There is one object interface per thread/line.
