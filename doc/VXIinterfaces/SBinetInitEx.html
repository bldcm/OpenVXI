Global platform initialization (extended version) of SBinet.

Parameters:
- log VXI Logging interface used for error/diagnostic logging, only used for the duration of this function call
diagLogBase - VXI Diagnostic Logging Base tag.
configParams - Map containing configuration parameters used to initialize SBinet. These parameters are the properties defined above. Once the function returns, this Map can safely be destroyed as it is the responsibility of SBinet to copy any data it might require after initialization. @result VXIinet_RESULT_SUCCESS on success
